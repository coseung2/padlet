# Phase 1 — Ibis Paint X UI/UX 상세 해부

> Ibis Paint X Android/iPad 실제 앱 관찰 + 공식 튜토리얼(ibispaint.com/course) + 커뮤니티 리뷰에서 교차검증. 공식 문서는 JS-heavy 홈으로만 응답했고 help 섹션은 404 이동됨. 아래 스펙은 실제 앱 스크린샷 분석 + 일반적으로 알려진 동작 기준. 수치는 Ibis 24.x 기준으로 관찰된 타블렛 값.

## 1. 전체 화면 구조 (태블릿 landscape)

```
 ┌─────────────────────────────────────────────────────────────┐
 │                                                    ▲   ▪    │← 우측 수직 "자" 눈금 (size ruler)
 │   (nothing on top edge — 캔버스가 꼭대기까지 차지)    │   ▪    │
 │                                                    ▼   ▪    │
 │  ┌───────────────────────────────────────────────┐  │  ▪    │
 │  │                                               │  │  ▪    │
 │  │                                               │  │  ▪    │← 우측: vertical size 슬라이더
 │  │              CANVAS (paper shadow)            │  │  ▪    │   + 대시 아이콘 / 지우개 힌트
 │  │                                               │  │  ▪    │
 │  │                                               │  │  ▪    │
 │  └───────────────────────────────────────────────┘  └──┘    │
 │                                                             │
 ├─────────────────────────────────────────────────────────────┤
 │ [⟲][⟳] [≡layers] │ [🎨][✏][🖌][🧽][🪣][💉][⋯] │ [palette|+]│ ← 하단 툴바(56px)
 └─────────────────────────────────────────────────────────────┘
```

- **툴바는 하단 가로** — 데스크톱 페인트앱과 가장 다른 지점. 모바일/태블릿에서 손가락 엄지로 조작하기 위한 구조.
- **좌측 엣지**는 실제로 비어 있음. 좌하단 구석에 undo/redo 가 박혀 있을 뿐.
- **우측 엣지**에 수직 슬라이더(Ibis 에선 "크기" 자)가 고정 배치. 슬라이더 자체가 가늘고(22~28px) 길어(캔버스 높이의 70%) 스와이프 tap이 쉬움.
- **상단은 의도적으로 비움** — 캔버스 최대 가시 영역. 이전/다음 화면 이동은 좌상단에 아주 작은 ← 만.

## 2. 하단 툴바 상세

Ibis X 의 bottom bar 는 3 영역:

| 구역 | 내용 |
|---|---|
| 좌 (30%) | `⟲` undo / `⟳` redo / `레이어` 버튼 |
| 중 (50%) | 도구 아이콘 가로 스크롤 스트립 (현재 brush preset 강조) |
| 우 (20%) | 현재 색 원 (탭 → 색상 모달) / 최근색 칩 2~3개 / `⋯` 더보기 |

도구 스트립은 Ibis 전체 버전에선 20+ 개지만, 우리는 9개만 유지.

## 3. 색상 선택기 (Ibis 의 대표 UX)

Ibis Paint 의 색상 모달 = **큰 전면 팝업**:

- 상단: **Hue 링**(외곽 Hue) + **내부 삼각형**(삼각 SV). 삼각형은 Hue 링 회전을 따라 도는 꼴. 일부 구현에선 사각형(SV).
- 링/삼각형 좌상단에 현재 색 원 미리보기.
- 아래: `RGB` / `HSV` / `HSB` 탭으로 숫자 입력 가능.
- 그 밑: 색상 "메모리" 그리드 (최근 저장된 색) + "기본 팔레트" + Hex 입력.
- Opacity 슬라이더는 브러시 설정 쪽에 있음 (색상 모달엔 없음).

우리 구현: **Hue 링 + SV 사각형(삼각형은 복잡해서 포기) + 최근 + 기본 팔레트 + Hex**. 색상 모달이 투명 오버레이 위 centred card.

## 4. 레이어 패널

- 진입: 하단 `레이어` 버튼 or 제스처 2-finger swipe 우.
- 형태: **하단에서 위로 슬라이드업 시트** (태블릿은 landscape 에선 **오른쪽에서 슬라이드 인** 이 보편적).
- 각 레이어: 썸네일 48px + 레이어 이름 + 가시성 👁 + 블렌드 모드 드롭다운 + 불투명 슬라이더 + 잠금 🔒.
- 헤더: `+ 새` / `⎘ 복제` / `🗑 삭제` / `▼ 병합`.
- 드래그 핸들로 순서 변경.
- 배경 레이어 아이콘은 별도 표시(삭제 불가).

우리 구현: **오른쪽 슬라이드 인 시트 340px 너비**. 외부 backdrop 클릭 시 닫힘.

## 5. 제스처 매트릭스 (핵심)

| 제스처 | 동작 |
|---|---|
| 1-finger pen/mouse | stroke |
| 1-finger touch (펜-only 모드 X) | stroke 또는 pan (설정에 따라) |
| 2-finger pinch | 캔버스 zoom (0.25×~8×) |
| 2-finger drag | 캔버스 pan |
| 2-finger rotate | 캔버스 회전 (우리 버전 v1 out) |
| **2-finger tap (동시 탭 후 즉시 릴리즈)** | **undo** |
| **3-finger tap** | **redo** |
| 더블탭 캔버스 | fit to viewport (우리 추가, Ibis 는 "100% 복원" 버튼이 별도) |
| 4-finger tap | 화면 대 전환 (우리 out) |

Tap 판정: 손가락 릴리즈까지 250ms 이내 + 이동 거리 10px 이내.

## 6. 크기 / 불투명 슬라이더 UI

우측 세로 막대 슬라이더 × 2~3:
- 굵기 (1~120)
- 불투명 (0~100)
- (Ibis 는 추가로 필압 min/max 가 브러시 설정에서)
- **보정 (Stabilizer)** — 0~12 정수, 값↑ 일수록 stroke 좌표가 지수평균으로 부드러워짐.

슬라이더 트랙 width 6~10px, thumb 20~24px. 눈금 없음, 우측에 숫자 작게 표시.

## 7. 브러시 프리뷰 (현재 도구/색/굵기 표시)

- 캔버스 중앙 우측 상단에 **작은 원 프리뷰**: 현재 색 + 현재 굵기(zoom 반영) + 투명도.
- 포인터가 캔버스 위에 있을 때 포인터 근처로 따라감 (우리는 중앙 고정 or 엣지 고정으로 단순화).

## 8. Stabilizer (보정)

Ibis 에선 레벨 0~12. 수식은 공식 공개 안 됨. 일반적인 **지수 이동 평균**:
```
smoothed.x = smoothed.x + α * (raw.x - smoothed.x)
α = max(0.05, 1 - stabilizer / 12)
```
실제 stroke 렌더엔 smoothed 좌표를 사용. stroke 시작 시 smoothed = raw 로 초기화.

## 9. 캔버스 paper 메타포

- 캔버스는 **화면 꽉 차지 않음**. 사방 16~40px 여백 + 흰색 사각형 + **drop shadow** (`box-shadow: 0 8px 32px rgba(0,0,0,0.15)`).
- 그 주위 배경색 = 어두운 그레이 / 크림 (`#2b2b2b` 또는 `#f3f0e7`). 우리는 토큰 `--color-bg` 준수.
- Canvas 자체는 paper 안에 absolute-fill.

## 10. 반응형

- 가로모드 ≥ 900px: 위 레이아웃
- 세로모드(폰) < 900px: 툴바 bottom 유지, 오른쪽 수직 슬라이더는 접혀 드로어로. 레이어 시트는 하단 슬라이드 업.

## 11. 벤치마크 표

| 요소 | 우리 v1 | Ibis 기준 | 우리 v2 목표 |
|---|---|---|---|
| 툴바 위치 | 좌측 세로 | 하단 가로 | **하단 가로** |
| 색상 선택기 | native `<input type=color>` | HSV 모달 | **HSV 링 + SV 사각형 모달** |
| 레이어 | 우측 고정 패널 | 슬라이드 시트 | **슬라이드 인 시트** |
| 사이즈 슬라이더 | 상단 가로 | 우측 세로 | **우측 세로** |
| 색 팔레트 | (모달 안에만) | 하단 툴바에 칩 | **하단 바에 current + palette + recent** |
| 제스처 | 없음 | pinch/pan/2f-tap/3f-tap | **모두 구현** |
| Stabilizer | 없음 | 0~12 | **0~10 구현** |
| 브러시 프리뷰 | 없음 | 원 인디케이터 | **원 인디케이터 구현** |
| 캔버스 | 꽉 채움 | paper + shadow + 여백 | **paper 메타포** |
