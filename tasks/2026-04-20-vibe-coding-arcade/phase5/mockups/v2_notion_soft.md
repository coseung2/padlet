## 장점

- 기존 design-system 99% 재사용 → 구현 비용 최소
- Notion 감성 일관성 · 기존 보드와 비주얼 통일
- 탭 S6 Lite 렌더 부담 최소 (gradient 없음, 단색 표면)
- 신규 토큰 7개로 제한 (phase4에서 명시된 규모와 동일)
- 승인 배지·반려 배너 재사용 가능

## 단점

- itch.io 대비 시각 자극 약함 — 초등 저학년에게는 "게임"보다 "문서"처럼 보일 수 있음
- 카드 한 장당 정보가 "thumbnail + 제목 + 작가 + 별·플레이수" 4줄로 큼 (v1 유사)
- 플레이 모달만 어두운 배경 → 브랜드 tone 이분 — 디자인 일관성 약간 깨짐 (but 몰입 목적 정당화 가능)

## 탭 S6 Lite 성능 이점

- `--shadow-card` 등 기존 시스템 4-layer shadow 이미 최적화됨
- gradient·blur 0 → 렌더 저비용
- 썸네일 위에 타이포 오버레이 없음 → 디코드 후 DOM 변화 없음 (CLS 0)
