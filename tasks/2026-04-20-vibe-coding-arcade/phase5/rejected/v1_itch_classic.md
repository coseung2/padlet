## 장점

- itch.io 벤치마크 충실 재현 · 학생에게 친숙
- 카드 그리드가 시각적으로 풍성
- 별점·플레이수 하단 라인 정보 밀도 적정

## 단점

- 배경 컬러풀 → Notion 기반 기존 design-system에서 이탈
- 썸네일 gradient 추출은 구현 비용 · 탭 S6 Lite 렌더 부담
- 카드당 메타 정보 4줄 → 탭 S6 Lite 세로 모드에서 카드 높이 증가 → TTI 위협
- 신규 토큰 ≥ 12개 (itch.io 고유 색상 재현)

## 탭 S6 Lite 성능 위험

- 썸네일 gradient 추출 클라 로직 추가 → §2c 번들 <500KB gzip 위협
- 카드 그리드 5열 기본 → 모바일 세로에서 1열 전환 시 스크롤 길어짐
