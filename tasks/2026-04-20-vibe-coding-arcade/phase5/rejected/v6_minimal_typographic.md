# Mockup v6 — Minimal Typographic

## 컨셉

썸네일 작게(64×48) · 제목·작가·태그 타이포 중심 · 밀집 리스트. Hacker News / 네이버 카페 목록 파생. 정보 밀도 극대화.

## 카탈로그 (S1)

```
┌────────────────────────────────────────────────────────────────┐
│ 🎮 학급 아케이드                      [신작][인기][추천][🎯]    │
├────────────────────────────────────────────────────────────────┤
│ [t]  테트리스 클론 #게임                  ⭐4.2  ▶23  by 김철수│
│ [t]  넌센스 퀴즈 #퀴즈                    ⭐3.8  ▶17  by 이영희│
│ [t]  고양이 워크 #시뮬                    ⭐4.9  ▶45  by 박지민│
│ [t]  스페이스 슛 #게임                    ⭐4.1  ▶31  by 정하늘│
│ [t]  만다라 아트 #아트                    ⭐4.5  ▶12  by 최수아│
│ ...                                                             │
│                                              [+ 새로 만들기] ● │
└────────────────────────────────────────────────────────────────┘
```

- 행당 높이 48px
- [t] = 썸네일 64×48
- 태그 pill inline

## 장점

- 정보 밀도 최고 — 한 화면에 20+개 작품 노출 가능
- 시각 자극 최소 → 교사·학부모 인식 무난
- 탭 S6 Lite 세로에서도 행 스크롤 자연스러움
- 번들 크기 최소

## 단점

- **초등 대상 "게임"에 전혀 맞지 않음** — 문서 목록처럼 보여서 진입 동기 약함
- 썸네일 주인공이 아님 → "바이브 코딩 결과물"의 시각적 성취 전달 실패
- 별점·리뷰 문화와 어울리지 않음 (Steam은 그리드, HN은 별점 없음)

## 판정

**부적합**. 교사 모더레이션 대시보드 승인 큐 리스트 영역에 국소 차용 가능(이미 v2가 일부 채택).
