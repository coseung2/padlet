## 장점

- 또래 사회적 상호작용 시각화 (작가 강조) — 학급 Steam 원래 목표(생산자-소비자 루프) 증폭
- 정사각 → 썸네일 160×120이 어울리지 않음 → **썸네일 비율 변경 필요**
- 인스타 감성 → 학생 친숙

## 단점

- **썸네일 1:1 재생성 필요** → 서버 Playwright 재작업 + §2c 성능 예산 재검토
- 작가 이모지 아바타 — 신규 엔티티·UI 확장(학생 프로필 이미지 시스템 부재) → v1 스코프 OUT과 충돌
- SNS 감성 → 학부모·교사 일각의 우려: "또래 경쟁·인기 콘테스트화"
- `reviewAuthorDisplay=named` (실명·번호) 기본값과 이모지 아바타 병행 → 정체성 혼란 (실명인데 이모지?)
