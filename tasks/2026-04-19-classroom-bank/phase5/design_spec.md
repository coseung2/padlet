# Design Spec — classroom-bank (v2 채택)

## 1. 선택된 변형
`mockups/v2` — 2-column 분할, 좌측 카드/우측 거래. 태블릿 중심 사용 시나리오에 최적.

## 2. 화면별 최종 레이아웃

### /my/wallet (학생 태블릿)
```

─────────────────────────────────────────
┌────────────────┬───────────────────┐
│ [좌측 컬럼]    │ [우측 컬럼]        │
│ 잔액 큰 숫자    │ 📜 최근 거래       │
```

- 모바일 768px 이하: stack

## 3. 사용된 토큰

### 기존 재사용
- 색상: `--color-bg/surface/text/border/accent/text-muted/text-faint`
- 상태 뱃지: `--color-status-reviewed-*` (입금 성공), `--color-status-returned-*` (출금/거부)
- 타이포: 기본 font-family, `--font-display` for 잔액 큰 숫자

### 신규 토큰 (2개)
| 토큰 | 값 | 용도 |
|---|---|---|
| `--color-bank-positive` | `#27a35f` | 입금/만기 (초록 계열, plant-active alias) |
| `--color-bank-negative` | `#c62828` | 결제/출금 (danger alias) |

이 2 토큰은 alias라 하드코딩 hex 아님. tokens_patch.json에 alias 구조로 저장.
