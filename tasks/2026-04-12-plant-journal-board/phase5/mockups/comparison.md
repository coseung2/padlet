# Shotgun Comparison — 학생 노선도 메인 뷰

## v1 — 서울 지하철 2호선 스타일 (수평 단색 라인)

```
[1]━━[2]━━[3]━━(4)━━[5]━━[6]━━ ... ━━[10]
 씨앗  떡잎  본잎   꽃   열매               완성

• 수평 SVG line, 노드는 원형 40px
• 현재 단계(4)는 1.5배 확대 + drop-shadow
• 완료 노드(1~3)는 채워진 동그라미
• 미래 노드(5~10)는 속빈 동그라미
• 전체 가로 스크롤, 모바일/태블릿 모두 동일
```

**장점**: 친숙함 최대, 개발 쉬움, iPad 9세대 성능 OK
**단점**: 시각적 흥미 약함 — 초등학생 대상으로는 드라이할 수 있음

## v2 — 성장 곡선(S-curve) + 식물 이모지 per 노드

```
          ·(5)🌿
        /       \
  (3)🌱         (6)🌺
  /              \
(1)🌰─(2)💧       (7)🍅 ...
```

• 곡선 Bezier path, 노드마다 식물 이모지를 크게 표시
• 단계별 이모지가 "성장하는" 시각화

**장점**: 교육적 재미, 단계 구분 직관
**단점**: 반응형 대응 복잡, SVG path 수학 필요, 길이 가변으로 설계 부담

## v3 — 수직 타임라인(모바일 퍼스트) + 상세 항상 노출

```
[1] 씨앗  ─┐
           │ 심은 날: 3/5  📷x3
[2] 떡잎  ─┤ 메모: 싹이 났어요!
           │
(3) 본잎  ─┤ <현재> 관찰 추가 버튼
           │
[4] 꽃    ─┘ 미래
...
```

• 세로 스크롤, 각 단계 오른쪽에 본인 기록 inline
• 하단 시트 없이도 완결

**장점**: 모바일에서 강력, 상세 접근성 최고
**단점**: "한눈에 전체 진행" 직관이 약함, 10단계 세로 스크롤 부담

## v4 — 카드 스택 + 슬라이더 (Instagram Stories 스타일)

```
< (1) 씨앗  [(2) 떡잎]  (3) 본잎 > 
          ┌──────────┐
          │  큰 카드   │
          │  사진/메모  │
          │  current   │
          └──────────┘
```

• 현재 단계만 크게 보여줌, 좌우 화살표/스와이프로 이전/다음
• 최상단 progress dots로 위치

**장점**: 시선 집중 최강, 사진이 풍성해 보임
**단점**: "전체 맥락" 파악이 어려움(progress dots만), 여러 단계 빠르게 비교 힘듦
