## AssessmentTake (학생, in-progress)
```
┌── ⏱ 29:45  남은 시간 ─────────────────── 저장됨 ─┐
├──────────────────────────────────────────────────┤
│  문항 1 / 5                                      │
│  세종대왕이 만든 문자는?                         │
│    (●) A 한글                                    │
│    ( ) B 한자                                    │
│    ( ) C 알파벳                                  │
│    ( ) D 가나다라                                │
│                                                  │
│  문항 2 / 5                                      │
│  ... (스크롤) ...                                │
│                                                  │
├──────────────────────────────────────────────────┤
│                           [ 제출 ]               │
└──────────────────────────────────────────────────┘
```
- 타이머 sticky top, 배경 흰색. < 5분: 빨강 + pulse.
- "저장됨" / "저장 중" 우상단 미세 텍스트.
- "제출" sticky bottom, 답 0개라도 비활성 아님 (경고 confirm 만).

## AssessmentGradebook (교사)
```
┌─ 단원평가 1차 — 성적부 ───────────────── [릴리스]─┐
│  제출 12/15명 · 평균 72점                        │
│                                                  │
│  ┌─────────┬─┬─┬─┬─┬─┬──┬──────┐                │
│  │ 이름    │1│2│3│4│5│총│확정  │                │
│  ├─────────┼─┼─┼─┼─┼─┼──┼──────┤                │
│  │ 홍길동  │🟢│🔴│🟢│⚪│🟢│4/5│[확정]│          │
│  │ 김영희  │🟢│🟢│🔴│🟢│🟢│4/5│[✓]  │          │
│  │ 이철수  │─│─│─│─│─│ - │ -   │ ← 미응시       │
│  └─────────┴─┴─┴─┴─┴─┴──┴──────┘                │
│  (이름 열 sticky, 매트릭스 가로 스크롤)          │
└──────────────────────────────────────────────────┘
```
- "릴리스" 버튼 우상단, 클릭 시 confirm → 확정된 학생만 releasedAt 일괄 SET.
- 학생별 "확정" 버튼 → finalize API.
- 매트릭스 셀 3색: 정답 녹 / 오답 빨 / 미응답 회.
